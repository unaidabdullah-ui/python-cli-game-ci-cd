## Overview

**Guess the Number** is a command-line game written in Python. The program randomly selects a number between 1 and 100, and the player tries to guess it. After each guess, the game provides feedback indicating if the guess was too low or too high until the correct number is guessed.

---

## Features

- Random number generation between 1 and 100.
- Unlimited attempts until the number is guessed correctly.
- Real-time feedback for too high or too low guesses.
- Input validation to ensure the player enters a valid number.
- Friendly emoji-enhanced messages for a fun experience.
